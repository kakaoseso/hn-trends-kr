# HN Trends KR Blog

This is the official blog for **HN Trends KR**, a project that curates and analyzes trending topics from Hacker News for Korean readers.

The blog is built with [Astro](https://astro.build) and hosted on GitHub Pages. It features daily summaries and technical insights from top technology discussions.

## Features
- **Daily Updates**: Automated posting of high-quality summarized content.
- **Korean Translation**: Expert-level translation and context for Korean audience.
- **Modern UI**: Clean, readable design using Noto Sans KR.

## Tech Stack
- **Framework**: Astro
- **Deployment**: GitHub Actions -> GitHub Pages
- **Automation**: Custom Python bots (JetSki / Gemini)
